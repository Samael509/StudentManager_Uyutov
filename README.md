# StudentManager

## Назначение программы
простое приложение для управления списком студентов — добавление и удаление записей из базы данных SQLite

## Реализованные функции
- Просмотр списка студентов
- Добавление нового студента
- Удаление выбранного студента
- сортировка по ID и имени
