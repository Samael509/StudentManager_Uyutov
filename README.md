# StudentManager

## Требования
- .NET Desktop Development workload
- Поддеркжа wpf (по дефолту она включена)
- .NET SDK оптимальная версия
- Microsoft.Data.Sqlite
- Sqlite (база данных)
- файл students.db должен сущестовать

## ОБРАТИТЬ ВНИМАНИЕ
- В MainWindow.xaml и MainWindow.xaml.cs должны быть правильные namespace'ы
- проверил NuGet пакеты, всё устанавлено. По новой устанавливать не надо

## Работа с базой данных
- путь: @"Data Source=C:\SOFT\students.db" или @"Data Source=C:\SOFT\authdemo.db"(можно изменить)

## Назначение программы
простое приложение для управления списком студентов — добавление и удаление записей из базы данных SQLite

## Реализованные функции
- Просмотр списка студентов
- Добавление нового студента
- Удаление выбранного студента
- сортировка по ID и имени
